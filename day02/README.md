## Password Strength Checker

Input: password

Check if it:

has at least 8 characters

contains numbers

contains uppercase & lowercase

Output: Weak / Medium / Strong

This program will check how secure the users password is and perform or print the following:
weak, medium or strong

Ask user to input their password which is string and check the following;

- Has at least 8 characters
- contains numbers
- containd uppercase or lowercase
